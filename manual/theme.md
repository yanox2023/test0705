# テーマ設定

商品ページにe-Gift Nowの購入導線を表示するための設定です。

::: warning
このページは仮原稿です。テーマエディタ上の正確な表示名は今後確認します。
:::

## テーマブロックを追加する

Shopify管理画面からテーマエディタを開き、商品ページテンプレートにe-Gift Nowのテーマブロックを追加します。

## 表示内容を確認する

対象商品ページで、以下を確認します。

- eギフト購入の導線が表示されること
- 通常購入とeギフト購入の区別が分かること
- スマートフォン表示でも操作できること

## 表示されない場合

商品ページに表示されない場合は、以下を確認します。

- 商品がeギフト対象に設定されているか
- テーマブロックが正しい商品テンプレートに追加されているか
- テーマを公開済みにしているか
- ブラウザキャッシュの影響がないか

詳細は[トラブルシューティング](/help/troubleshooting)を確認してください。
