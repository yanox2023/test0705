# はじめての設定

e-Gift Nowをインストールしたあと、最短で動作確認するための初期設定手順です。

::: warning
このページは仮原稿です。実際の画面名、操作手順、スクリーンショットは今後整理します。
:::

## 1. アプリをインストールする

Shopify管理画面からe-Gift Nowをインストールします。

インストール後、アプリ管理画面を開き、初期設定に必要な項目を確認します。

## 2. 初期設定を行う

アプリ管理画面で、eギフト注文に必要な基本設定を行います。

- 受取ページの表示内容
- 受取人への案内文
- 贈る人の名前やメッセージの設定
- お届け日時の入力設定
- 通知メールの設定

## 3. 商品をeギフト対応にする

eギフトとして販売したい商品を選び、e-Gift Nowの対象商品として設定します。

設定後、商品ページでeギフト購入用の表示が出ることを確認します。

## 4. テーマにブロックを追加する

Shopifyのテーマエディタを開き、商品ページにe-Gift Nowのテーマブロックを追加します。

対象商品ページで、eギフト購入の導線が表示されることを確認します。

## 5. テスト注文する

テスト用の商品で注文を作成し、以下を確認します。

- eギフト注文として注文情報が保存されること
- ギフトURLが発行されること
- 受取人が住所を入力できること
- 住所入力後に注文情報やタグが更新されること
